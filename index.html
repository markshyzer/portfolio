Checkers

Demo - Code

A 2-player local version of the classic board game, with full game logic implemented in vanilla Javascript. Click a piece to see all available legal moves, then click again to move, jump, or replace the piece and try another.

HTML5, CSS3, Javascript

Chaptr

Demo - Code

Chaptr is designed to manage all aspects of your book club. Users can create a group and invite members, who can schedule meetings and discuss and rate readings. Chaptr uses the Google Books API to provide book data, making it easy to look up and recommend the next book on your reading list.

MatereializeCSS, Python, Django
requests

Puzzle Project

Demo - Code

This is a multi-player puzzle app, where players can collaborate to complete a jigsaw-style puzzle together in real time in a public or private room. Each room also features its own chat to stay connected while you play.

MongoDB, Express, React, Node (MERN Stack)
Socket.io, react-draggable

Puzzle Project

Demo - Code

Connecto is a geometry puzzle game: connect all the dots without passing through any dot more than once to win. Earn a trophy for each of the three puzzles you complete in this demo.
