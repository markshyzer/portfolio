Sony Remote

Use the remote - See the code

My remote broke, so I built a new one. Designed with a simple and familiar UI, this virtual remote can be used to control any post-2016 Sony Bravia TV from any device on the same network. Select which remote buttons you need and switch quickly from a customized compact view to a full standard remote.

HTML5, CSS3, Javascript, IRCC-IP

Checkers

A 2-player local version of the classic board game, with full game logic implemented in vanilla Javascript. Click a piece to see all available legal moves, then click again to move, jump, or replace the piece and try another.
